Tower Defense: Pure Math Edition
Welcome to Tower Defense: Pure Math Edition, a high-performance, procedurally rendered strategy game built with Unity. This project was developed as a midterm requirement for Applied Math, focusing on implementation without Unity's standard physics engine, favoring custom mathematical solutions for rendering and collisions.

Gameplay Mechanics
- Strategic Tower Building: Place and manage towers to defend against waves of enemies.
- Dynamic Wave System: Escalating difficulty with a custom "WaveManager" handling enemy spawning patterns.
- Resource Management: Earn gold by defeating enemies to build more powerful defenses.
- Health & Armor Logic: Multi-layered health systems for both the player and enemies.

Visuals & UI
- Animated Game HUD: Smoothly lerping health bars, gold counters, and wave indicators.
- Full Menu System: Interactive Main Menu, Settings (Audio/Visual), and Pause menus.

Built With
- Engine: Unity 6 (6000.3.2f1)
- Language: C#
- Input: Unity New Input System
- Rendering: Universal Render Pipeline (URP)
